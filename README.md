# My-First--FIFA-U17-2025-EVENTS
SQL Project: cleaned FIFA U17 event data(goals, red/yellow cards). Phase 1

This project analyzes match events from the FIFA U17 2025 dataset. It includes player actions such as goals, yellow cards, and red cards. The dataset was cleaned and structured using SQL to separate events into columns for analysis.

Columns include: PlayID, AthleteID, AthleteName, PlayOrder, Period, EventType, TeamID, SeasonType, YellowCard, RedCard, Goal, ClockDisplay.

Phase 1 focuses on cleaning the data and separating events using SQL CASE statements. Future phases may include aggregations and visualizations.
